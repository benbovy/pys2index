pys2index Documentation
=======================

Python / NumPy compatible geographical index based on s2geometry_.

This project doesn't provide Python wrappers for the whole s2geometry library.
Instead, it aims to provide some index wrappers with an API similar to
``scipy.spatial.cKDTree`` or ``sklearn.neighbors`` indexes.

.. _s2geometry: https://s2geometry.io
